W celu skorzystania z aplikacji:
Uruchom plik MyApplication.exe 
lub
Użyj programu Visual Studio 2013. 

W celu otwarcia programu w Visual Studio 2013.
Otwórz plik poprzez menu programu (Menu -> OpenProject).
lub
Dwukrotnie kliknij lewym przyciskiem myszy na plik MyApplication.sln

In order to use the application:
Run MyApplication.exe
or
Use Visual Studio 2013.

To open the program in Visual Studio 2013.
Open the file via the program menu (Menu -> OpenProject).
or
Double click the left mouse button on the file MyApplication.sln
